هذا البرنامج مخصص للمهندسين أصحاب الخبرة الذين يبحثون عن انتقال حقيقي ومدروس إلى مجال DevOps و Kubernetes.

لـ Developers و DevOps Engineers و Cloud Engineers

مسار احترافي مكثف لمدة 12 أسبوعًا ينقلك من كتابة Code فقط إلى امتلاك القدرة على Build و Deploy و Operate أنظمة Production حقيقية

مناسب لك إذا كنت:

  • Developer بخبرة +3 سنوات وتريد الانتقال إلى DevOps بمستوى احترافي
  • DevOps Engineer يعمل بالفعل ولكن يريد إتقان Kubernetes والعمل بثقة في Production
  • Cloud Engineer يريد التحكم الكامل في Lifecycle للتطبيق من Build حتى Deployment والتشغيل

ليس مناسبًا لك إذا كنت

  • مبتدئًا تمامًا في البرمجة أو في Linux
  • لم يسبق لك استخدام Git أو Terminal أو كتابة Scripts بسيطة
  • لا تستطيع الالتزام بحضور 4 ساعات أسبوعيًا Live Sessions بالإضافة إلى وقت كافٍ لتنفيذ المشاريع العملية
  • تبحث عن محتوى نظري فقط دون تطبيق عملي مكثف يحاكي بيئات Production
  • هذا البرنامج يتطلب خلفية تقنية سابقة واستعدادًا حقيقيًا للالتزام والعمل الجاد طوال مدة المسار
